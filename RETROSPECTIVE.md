# RETROSPECTIVE

## 기간
- 2026-04 ~ 2026-05

## 목표 대비 결과
- 목표: 경찰고시 판례 학습을 위한 AI 기능을 서버 없이 단말 내부에서 완결 실행
- 결과: iOS 앱에서 검색, 요약, OX 퀴즈, 오답 복습까지 온디바이스 파이프라인으로 구현

## 잘한 점
- 온디바이스 우선 설계로 네트워크/백엔드 장애 지점을 제거함
- LLM 실패 시 Rule-based fallback을 둬서 사용자 흐름이 끊기지 않게 함
- CI에 린트/테스트/보안 스캔을 포함해 PR 게이트를 자동화함

## 아쉬운 점
- 초기에는 백엔드 전제를 일부 문서에서 완전히 제거하지 못해 기준 해석 혼선이 있었음
- 관측성 지표(지연/오류율) 정의가 구현보다 늦게 정리됨
- 테스트가 도메인 핵심 경로 대비 아직 부족함

## 액션 아이템
- 온디바이스 헬스체크를 테스트 항목으로 정기 실행
- 실패 원인 분류(모델 로드/추론/파싱) 기반 로그 규약 고도화
- 릴리스마다 3분 데모 영상과 체크리스트를 동일 템플릿으로 관리

- 성능 지표 수집 주기와 경고 임계치 문서화
- App Store 심사 대응 문서 표준화
