# STACK112 개인정보처리방침

**최종 수정일**: 2026년 5월 12일
**앱 이름**: STACK112 (이하 "본 앱")
**개발자**: 임재현 (개인 개발자)
**연락처**: <개발자 이메일 주소 기입>

본 방침은 사용자가 본 앱을 이용함에 있어 발생할 수 있는 개인정보 처리에 관한 사항을
투명하게 안내하기 위해 작성되었습니다.

---

## 1. 한 줄 요약

> **본 앱은 어떠한 개인정보도 외부 서버로 전송·수집·저장하지 않습니다.**
> 모든 데이터(스캔 이미지, 학습 기록, 오답노트, 저장 판례)는 사용자의 기기 내부에만
> 저장되며, 본 앱은 외부 통신을 수행하지 않습니다.

---

## 2. 수집하는 정보

본 앱은 다음 정보를 **수집하지 않습니다**.

- 이름, 이메일, 전화번호, 주소 등 식별 정보
- 위치 정보 / GPS
- 광고 식별자 (IDFA)
- 사용 로그 / 크래시 리포트 / 분석 데이터
- 결제 정보
- 사용자가 입력한 텍스트, 검색어, 학습 내용

App Store Connect 의 "App Privacy" 신고에서도 **Data Not Collected** 로 분류됩니다.

---

## 3. 기기 내부에서 처리하는 정보

본 앱은 아래 정보를 사용자 기기 내부에서만 처리하며, 외부로 전송하지 않습니다.

| 항목 | 처리 방식 | 저장 위치 |
| --- | --- | --- |
| 사진 라이브러리 이미지 | 사용자가 직접 선택한 이미지를 OCR(글자 인식)에 사용 | 기기 내부 (SwiftData) |
| OCR 인식 결과 텍스트 | 판례 메모 작성에 사용 | 기기 내부 (SwiftData) |
| 학습 기록 / 오답노트 / 저장 판례 | 학습 통계 표시에 사용 | 기기 내부 (SwiftData / UserDefaults) |
| AI 요약·비교·O/X 출력 | 본 앱에 번들된 온디바이스 LLM(Llama 3.2 1B) 으로 생성 | 기기 내부 메모리 (영구 저장 안 함) |

이 모든 처리는 **인터넷 연결 없이** 동작하며, 본 앱은 사용자 동의 없이 외부 서버에
어떠한 데이터도 송신하지 않습니다.

---

## 4. 권한 사용 안내

| 권한 | 사용 목적 | 거부 시 영향 |
| --- | --- | --- |
| 사진 라이브러리 (NSPhotoLibraryUsageDescription) | OCR 을 위해 사용자가 직접 선택한 이미지 1장을 읽음 | OCR 기능 사용 불가. 다른 기능은 정상 동작 |

본 앱은 카메라, 마이크, 위치, 연락처, 캘린더, 알림 등 **다른 어떤 권한도 요청하지 않습니다**.

---

## 5. 제 3자 제공 / 위탁

본 앱은 제 3자에게 정보를 제공하거나 처리를 위탁하지 않습니다.
광고 SDK, 분석 SDK, 크래시 리포트 SDK 등 외부 라이브러리를 통한 데이터 송신도
일절 없습니다.

---

## 6. 데이터 삭제

사용자는 본 앱 **설정 → 데이터 관리** 에서 언제든지 다음 항목을 삭제할 수 있습니다.

- 스캔한 판례 메모 전체 삭제
- 오답노트 전체 삭제
- 저장한 판례 전체 삭제

앱 자체를 삭제하면 위 모든 데이터가 기기에서 완전히 제거됩니다.
