# Model Card: On-device Legal Study Assistant

## 모델 개요
- 목적: 경찰고시 판례 학습 보조(요약, OX 생성, 검색 보조)
- 실행 환경: iOS 온디바이스
- 기본 구성: llama.cpp 기반 로컬 LLM + rule-based fallback

## 입력/출력
- 입력: 판례 텍스트, OCR 결과, 사용자 검색 질의
- 출력: 판례 요약, 핵심 쟁점, OX 문제, 간단 비교 분석

## 안전장치
- 모델 로드 실패 시 fallback 엔진 전환
- 파싱 실패 시 보수적 출력 생성
- 근거 중심 출력 유도 프롬프트 사용

## 제한사항
- 법률 자문 도구가 아님
- 희소 케이스에서 요약 일관성 변동 가능
- 온디바이스 자원(메모리/발열)에 민감

## 평가
- 정량: 앱 테스트 및 CI 통과 여부
- 정성: 핵심 사용자 플로우(OCR -> 요약 -> OX -> 복습) 수동 검증

## 운영 가이드
- 릴리스마다 헬스체크(HC-1~3) 수행
- fallback 비율/지연/크래시 지표를 관측
