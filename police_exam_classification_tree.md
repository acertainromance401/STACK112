
# 경찰 시험 핵심 분류 트리 정리

# 1. 경찰 시험의 기본 사고 구조

경찰 시험은 대부분 다음 구조로 문제를 풂.

사실관계 입력
→ 법적 유형 분류
→ 적용 법리 선택
→ 예외 검토
→ 정답 도출

핵심은:
- 단순 암기가 아니라
- “어떤 유형인지 빠르게 분류”하는 것.

# 2. 경찰 시험 전체 메인 분류 트리

문제
│
├─ 범죄 성립 문제인가?
│   ├─ 형법
│   └─ 특별법
│
├─ 수사 절차 문제인가?
│   └─ 형사소송법
│
├─ 국가 권력 제한 문제인가?
│   └─ 헌법
│
└─ 경찰 작용 문제인가?
    └─ 경찰학

시험장에서 가장 먼저 해야 하는 것:
- “이 문제가 어느 과목 구조인지” 분류하기.

# 3. 형법 핵심 분류 트리

형법은 “범죄 성립 단계”로 분류함.

행위 발생
│
├─ 구성요건 해당성
│   ├─ 고의범
│   ├─ 과실범
│   ├─ 결과범
│   └─ 거동범
│
├─ 위법성
│   ├─ 정당방위
│   ├─ 긴급피난
│   ├─ 피해자 승낙
│   └─ 자구행위
│
├─ 책임
│   ├─ 책임능력
│   ├─ 금지착오
│   └─ 기대가능성
│
└─ 범죄 형태
    ├─ 미수
    ├─ 불능미수
    ├─ 예비
    ├─ 교사범
    └─ 공동정범

# 4. 형법 실전 압축 트리

사람 죽음?
│
├─ 고의 있음 → 살인
├─ 과실 → 과실치사
├─ 실행 실패 → 미수
└─ 애초 불가능 → 불능미수

핵심:
- 결과보다
- 행위자의 “인식과 의도”가 중요.

# 5. 재산범죄 분류 트리

재산 이동 발생
│
├─ 몰래 가져감 → 절도
│
├─ 폭행·협박 사용
│   └─ 강도
│
├─ 속여서 받음
│   └─ 사기
│
├─ 맡긴 돈 안 돌려줌
│   └─ 횡령
│
└─ 임무 위반으로 손해
    └─ 배임

핵심 구별 기준:
- 피해자의 의사 개입 여부
- 재산 보관 관계 존재 여부
- 폭행·협박 존재 여부

# 6. 위법성 조각 분류 트리

위법한 행동처럼 보임
│
├─ 현재 공격 방어?
│   └─ 정당방위
│
├─ 위험 회피?
│   └─ 긴급피난
│
├─ 피해자 동의?
│   └─ 피해자 승낙
│
└─ 권리 스스로 실현?
    └─ 자구행위

# 7. 정당방위 vs 긴급피난

정당방위
상대방의 공격
↓
그 공격 방어

예:
- 강도 제압
- 폭행 방어

긴급피난
긴급한 위험
↓
더 큰 피해 회피

예:
- 화재 탈출 위해 문 파손

# 8. 형사소송법 핵심 분류 트리

형소법은:
- 국가 권력의 적법성 판단 과목.

수사 행위 발생
│
├─ 강제처분?
│   ├─ YES → 영장 필요 원칙
│   └─ NO → 임의수사 가능
│
├─ 체포?
│   ├─ 현행범체포
│   ├─ 긴급체포
│   └─ 영장체포
│
├─ 압수·수색?
│   ├─ 영장 있음
│   └─ 예외 여부 검토
│
└─ 증거 사용 가능?
    ├─ 적법수집
    └─ 위법수집

# 9. 영장주의 핵심 트리

강제처분인가?
│
├─ YES
│   ├─ 영장 있음 → 적법 가능
│   └─ 영장 없음 → 위법 가능
│
└─ NO
    └─ 임의수사 가능

핵심 함정:
- 임의 제출
- 동의 여부
- 긴급 상황 예외

# 10. 증거능력 분류 트리

증거 사용 가능?
│
├─ 적법 수집?
│   ├─ YES
│   └─ NO → 위법수집증거배제
│
├─ 전문증거?
│   ├─ YES → 예외 필요
│   └─ NO
│
└─ 자백?
    ├─ 강요 여부
    └─ 보강증거 여부

형소법 핵심:
- 절차 위반 시 증거능력 제한 가능.

# 11. 헌법 핵심 분류 트리

헌법은:
- “어떤 기본권 침해인지” 분류하는 과목.

국가 제한 발생
│
├─ 말/인터넷 제한
│   └─ 표현의 자유
│
├─ 직업 제한
│   └─ 직업의 자유
│
├─ 위치추적/감시
│   └─ 사생활의 자유
│
├─ 차별 발생
│   └─ 평등권
│
└─ 신체 구속
    └─ 신체의 자유

# 12. 실제 시험장에서의 사고 흐름

문제 읽음
↓
핵심 키워드 탐지
↓
유형 분류
↓
관련 판례 연결
↓
예외 검토
↓
정답 선택

# 13. 경찰 시험의 실제 본질

경찰 시험은:
- 단순 암기 시험이 아니라
- “패턴 인식 + 유형 분류 시험”에 가까움.

결국:

수천 개 문제
↓
50~100개 핵심 패턴으로 압축
↓
패턴 변형 반복

구조로 출제됨.

# 14. 가장 효율적인 공부 방식

추천 공부 흐름:

기출 풀이
↓
유형 태그 달기
↓
분류 트리 연결
↓
예외 정리
↓
함정 기록

핵심은:
- 문제를 “외우는 것”이 아니라
- “어떤 유형인지 즉시 분류 가능하게 만드는 것”.

# 15. 10초 과목 분류 체크리스트

문제를 읽고 아래 순서로 빠르게 체크.

1. 행위자 처벌 여부가 핵심인가?
- YES: 형법/특별법 우선.

2. 수사기관 행위의 적법성이 핵심인가?
- YES: 형사소송법 우선.

3. 국가가 국민의 자유를 제한하는 장면인가?
- YES: 헌법 우선.

4. 경찰 조직/작용/행정작용 기준을 묻는가?
- YES: 경찰학 우선.

헷갈릴 때 기준:
- 정답 문장이 “처벌 가능/불가” 중심이면 형법.
- 정답 문장이 “증거능력/영장/절차 위반” 중심이면 형소법.
- 정답 문장이 “과잉금지/본질침해/평등심사” 중심이면 헌법.

# 16. 과목별 판단 질문, 함정, 기출 키워드

## 16-1. 형법

판단 질문:
- 이 사안은 구성요건, 위법성, 책임 중 어디에서 갈리는가?
- 결과보다 행위자의 인식(고의/과실)이 쟁점인가?

자주 나오는 함정:
- 결과만 보고 고의범으로 단정.
- 정당방위와 긴급피난을 공격 주체 기준으로 구별하지 못함.
- 불능미수와 미수를 실행 가능성 기준으로 나누지 못함.

기출 키워드 트리거:
- 살인의 고의, 미필적 고의, 과실, 상당인과관계, 불능미수, 공동정범.

## 16-2. 형사소송법

판단 질문:
- 수사기관 행위가 강제처분인가 임의수사인가?
- 영장주의 예외가 인정되는 사실관계가 있는가?

자주 나오는 함정:
- 임의동행, 임의제출을 곧바로 적법으로 결론.
- 위법수집증거배제와 2차 증거의 예외 논리를 누락.
- 전문법칙 예외를 자백법칙과 혼동.

기출 키워드 트리거:
- 긴급체포, 현행범, 압수수색, 임의제출물, 위법수집증거배제, 전문증거.

## 16-3. 헌법

판단 질문:
- 제한 대상 기본권이 무엇인가?
- 심사 기준이 과잉금지원칙인지 평등심사인지가 핵심인가?

자주 나오는 함정:
- 기본권 주체성 검토 없이 바로 위헌/합헌 결론.
- 법률유보와 비례원칙의 심사 순서를 섞음.
- 평등권 사안에서 차별 목적의 합리성 검토 누락.

기출 키워드 트리거:
- 과잉금지, 본질적 내용 침해, 명확성 원칙, 평등원칙, 표현의 자유, 직업의 자유.

## 16-4. 경찰학

판단 질문:
- 경찰권 발동의 법적 근거와 한계가 제시되었는가?
- 즉시강제/행정조사/행정지도 중 무엇인가?

자주 나오는 함정:
- 경찰권 발동 목적의 정당성만 보고 수단의 상당성 검토 누락.
- 질서유지와 범죄수사의 경계를 혼동.
- 재량행위와 기속행위를 사실관계 없이 단정.

기출 키워드 트리거:
- 경찰비례원칙, 즉시강제, 경찰책임의 원칙, 위험방지, 재량통제.

# 17. 오답 줄이는 3단계 태깅 규칙

문제 1개를 풀 때 아래 3개 태그를 항상 붙인다.

1. 과목 태그
- 형법/형소법/헌법/경찰학

2. 쟁점 태그
- 예: 영장주의, 정당방위, 과잉금지, 위법수집증거배제

3. 함정 태그
- 예: 고의 판단 누락, 예외요건 누락, 절차 선후관계 오판

활용법:
- 같은 함정 태그가 3회 이상 반복되면 약점 영역으로 고정.
- 약점 영역은 판례 5개 + OX 20문제로 단기 반복.

# 18. 실전 적용용 미니 결정 트리

문제에서 경찰/검사/사법경찰관 행위가 보이면
→ 형소법 먼저 의심
→ 강제처분 여부 확인
→ 영장 또는 예외요건 체크

문제에서 기본권 제한 사유가 보이면
→ 헌법 먼저 의심
→ 제한 기본권 특정
→ 비례원칙 4요소 또는 평등심사 적용

문제에서 처벌 조문과 고의/과실이 보이면
→ 형법 먼저 의심
→ 구성요건, 위법성, 책임 순서로 점검

# 19. 마지막 정리

고득점 포인트는 양이 아니라 속도와 정확도다.

- 속도: 10초 안에 과목 분류.
- 정확도: 예외요건까지 확인 후 결론.
- 유지: 오답을 태그 기반으로 누적 관리.

이 문서는 암기 노트가 아니라 분류 엔진으로 사용해야 한다.

# 20. 형사소송법 전용 상세 트리 (실전판)

형소법 문제는 아래 4단계 순서로 풀면 오답이 줄어든다.

1. 처분 성격 확정: 임의수사 vs 강제처분
2. 영장 원칙 확인: 영장 필요 여부
3. 예외 요건 점검: 긴급성, 현행성, 동의, 사후영장
4. 증거능력 판단: 위법수집 여부, 전문법칙, 자백법칙

## 20-1. 임의수사 vs 강제처분 1차 분기

수사기관 행위 발생
│
├─ 상대방 의사에 반해 신체/주거/물건/정보를 실질 제한?
│   ├─ YES → 강제처분 가능성 높음
│   └─ NO  → 임의수사 가능성
│
└─ 사실상 거절 불가능한 압박이 있었는가?
    ├─ YES → 형식상 임의라도 강제처분 취급 가능
    └─ NO  → 임의수사 유지 가능

실전 포인트:
- 명칭보다 실질을 본다. "임의동행" 표기가 있어도 사실상 강제면 위법 가능.

## 20-2. 체포 파트 분류 트리

피의자 신체 확보 필요
│
├─ 범행 직후/현장에서 범죄와의 결합이 강함?
│   └─ 현행범체포 검토
│
├─ 영장 받을 시간적 여유 없음 + 중대성/필요성 높음?
│   └─ 긴급체포 검토
│
└─ 위 사유 부족
    └─ 원칙: 체포영장 필요

체포 문제 10초 체크:
- 체포 사유(혐의 소명)
- 필요성(도망/증거인멸 우려)
- 절차 준수(고지, 변호인 조력권, 사후 절차)

## 20-3. 압수·수색 파트 분류 트리

증거 확보 필요
│
├─ 영장 있음?
│   ├─ YES → 집행 범위/장소/시간 일치 확인
│   └─ NO  → 예외 요건 엄격 심사
│
├─ 임의제출물인가?
│   ├─ YES → 제출 주체 권한/진정한 동의 확인
│   └─ NO  → 영장 없는 강제수집이면 위법 위험 높음
│
└─ 전자정보/디지털 매체인가?
    └─ 탐색 범위 과잉 여부 추가 점검

핵심 함정:
- "동의했다"는 문구만 보고 종료하지 말고 동의권자 적격을 확인.
- 영장 범위를 벗어난 별건 탐색은 위법 포인트.

## 20-4. 위법수집증거배제 트리

증거 제출됨
│
├─ 수집절차 위법?
│   ├─ NO → 원칙적으로 증거능력 인정 방향
│   └─ YES
│       ├─ 위법 정도 중대 + 절차 보호 목적 침해 큼
│       │   └─ 배제 가능성 높음
│       └─ 경미 위반 + 인과 희석/예외 사유
│           └─ 예외 인정 가능성 검토
│
└─ 2차 증거(파생증거)인가?
    ├─ YES → 1차 위법과 인과관계 단절 여부 심사
    └─ NO

실전 포인트:
- 결론만 쓰지 말고 "위법성-보호목적-인과관계" 3요소를 문장에 넣어야 득점 안정.

## 20-5. 전문증거 트리 (진술증거)

진술 내용이 증거로 제출됨
│
├─ 법정 외 진술인가?
│   ├─ NO → 전문법칙 문제 약함
│   └─ YES
│       ├─ 원진술자 반대신문 보장?
│       ├─ 특신상태/성립의 진정 등 요건 충족?
│       └─ 조문상 예외 해당?

핵심 함정:
- "수사기관 작성"이라는 이유만으로 일괄 배제/일괄 인정하지 말 것.
- 증거서류 유형별 요건을 분리해서 본다.

## 20-6. 자백법칙 트리

피고인 자백 존재
│
├─ 임의성 의심(강압/회유/장기구금 등)?
│   ├─ YES → 임의성 부정 시 증거능력 부정 가능
│   └─ NO
│
└─ 자백이 유일 증거인가?
    ├─ YES → 보강증거 필요
    └─ NO

실전 포인트:
- 자백 문제는 임의성 + 보강증거를 항상 한 세트로 확인.

# 21. 형사소송법 고난도 함정 모음

1. "임의"라는 단어에 속는 경우
- 표시가 임의여도 현실적으로 거부가 불가능하면 강제처분 판단 가능.

2. 영장주의 예외를 넓게 해석하는 경우
- 예외는 엄격해석이 기본. 사안의 긴급성과 필요성을 구체 사실로 써야 함.

3. 위법수집증거에서 파생증거 판단 누락
- 1차 증거만 보고 끝내면 감점. 2차 증거 인과관계까지 검토 필요.

4. 전문법칙과 자백법칙 혼동
- 전문법칙은 진술증거 유형과 반대신문 보장 구조, 자백법칙은 임의성과 보강증거 구조.

# 22. 형사소송법 30초 답안 템플릿

아래 4문장 구조로 쓰면 시간 압박에서 안정적이다.

1. 쟁점 특정
- "이 사안의 쟁점은 영장 없는 압수의 적법성과 그에 따른 증거능력이다."

2. 기준 제시
- "강제처분은 영장주의가 원칙이고, 예외는 엄격한 요건 하에서만 허용된다."

3. 사안 포섭
- "본 사안은 동의권자 적격이 불분명하고 범위를 넘는 탐색이 있어 위법성이 크다."

4. 결론
- "따라서 해당 1차 증거는 배제 가능성이 높고, 파생증거 역시 인과관계 단절이 없으면 제한된다."
