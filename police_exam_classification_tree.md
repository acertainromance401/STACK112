
# 경찰 시험 핵심 분류 트리 정리

# 1. 경찰 시험의 기본 사고 구조

경찰 시험은 대부분 다음 구조로 문제를 풂.

사실관계 입력
→ 법적 유형 분류
→ 적용 법리 선택
→ 예외 검토
→ 정답 도출

핵심은:
- 단순 암기가 아니라
- “어떤 유형인지 빠르게 분류”하는 것.

# 2. 경찰 시험 전체 메인 분류 트리

문제
│
├─ 범죄 성립 문제인가?
│   ├─ 형법
│   └─ 특별법
│
├─ 수사 절차 문제인가?
│   └─ 형사소송법
│
├─ 국가 권력 제한 문제인가?
│   └─ 헌법
│
└─ 경찰 작용 문제인가?
    └─ 경찰학

시험장에서 가장 먼저 해야 하는 것:
- “이 문제가 어느 과목 구조인지” 분류하기.

# 3. 형법 핵심 분류 트리

형법은 “범죄 성립 단계”로 분류함.

행위 발생
│
├─ 구성요건 해당성
│   ├─ 고의범
│   ├─ 과실범
│   ├─ 결과범
│   └─ 거동범
│
├─ 위법성
│   ├─ 정당방위
│   ├─ 긴급피난
│   ├─ 피해자 승낙
│   └─ 자구행위
│
├─ 책임
│   ├─ 책임능력
│   ├─ 금지착오
│   └─ 기대가능성
│
└─ 범죄 형태
    ├─ 미수
    ├─ 불능미수
    ├─ 예비
    ├─ 교사범
    └─ 공동정범

# 4. 형법 실전 압축 트리

사람 죽음?
│
├─ 고의 있음 → 살인
├─ 과실 → 과실치사
├─ 실행 실패 → 미수
└─ 애초 불가능 → 불능미수

핵심:
- 결과보다
- 행위자의 “인식과 의도”가 중요.

# 5. 재산범죄 분류 트리

재산 이동 발생
│
├─ 몰래 가져감 → 절도
│
├─ 폭행·협박 사용
│   └─ 강도
│
├─ 속여서 받음
│   └─ 사기
│
├─ 맡긴 돈 안 돌려줌
│   └─ 횡령
│
└─ 임무 위반으로 손해
    └─ 배임

핵심 구별 기준:
- 피해자의 의사 개입 여부
- 재산 보관 관계 존재 여부
- 폭행·협박 존재 여부

# 6. 위법성 조각 분류 트리

위법한 행동처럼 보임
│
├─ 현재 공격 방어?
│   └─ 정당방위
│
├─ 위험 회피?
│   └─ 긴급피난
│
├─ 피해자 동의?
│   └─ 피해자 승낙
│
└─ 권리 스스로 실현?
    └─ 자구행위

# 7. 정당방위 vs 긴급피난

정당방위
상대방의 공격
↓
그 공격 방어

예:
- 강도 제압
- 폭행 방어

긴급피난
긴급한 위험
↓
더 큰 피해 회피

예:
- 화재 탈출 위해 문 파손

# 8. 형사소송법 핵심 분류 트리

형소법은:
- 국가 권력의 적법성 판단 과목.

수사 행위 발생
│
├─ 강제처분?
│   ├─ YES → 영장 필요 원칙
│   └─ NO → 임의수사 가능
│
├─ 체포?
│   ├─ 현행범체포
│   ├─ 긴급체포
│   └─ 영장체포
│
├─ 압수·수색?
│   ├─ 영장 있음
│   └─ 예외 여부 검토
│
└─ 증거 사용 가능?
    ├─ 적법수집
    └─ 위법수집

# 9. 영장주의 핵심 트리

강제처분인가?
│
├─ YES
│   ├─ 영장 있음 → 적법 가능
│   └─ 영장 없음 → 위법 가능
│
└─ NO
    └─ 임의수사 가능

핵심 함정:
- 임의 제출
- 동의 여부
- 긴급 상황 예외

# 10. 증거능력 분류 트리

증거 사용 가능?
│
├─ 적법 수집?
│   ├─ YES
│   └─ NO → 위법수집증거배제
│
├─ 전문증거?
│   ├─ YES → 예외 필요
│   └─ NO
│
└─ 자백?
    ├─ 강요 여부
    └─ 보강증거 여부

형소법 핵심:
- 절차 위반 시 증거능력 제한 가능.

# 11. 헌법 핵심 분류 트리

헌법은:
- “어떤 기본권 침해인지” 분류하는 과목.

국가 제한 발생
│
├─ 말/인터넷 제한
│   └─ 표현의 자유
│
├─ 직업 제한
│   └─ 직업의 자유
│
├─ 위치추적/감시
│   └─ 사생활의 자유
│
├─ 차별 발생
│   └─ 평등권
│
└─ 신체 구속
    └─ 신체의 자유

# 12. 실제 시험장에서의 사고 흐름

문제 읽음
↓
핵심 키워드 탐지
↓
유형 분류
↓
관련 판례 연결
↓
예외 검토
↓
정답 선택

# 13. 경찰 시험의 실제 본질

경찰 시험은:
- 단순 암기 시험이 아니라
- “패턴 인식 + 유형 분류 시험”에 가까움.

결국:

수천 개 문제
↓
50~100개 핵심 패턴으로 압축
↓
패턴 변형 반복

구조로 출제됨.

# 14. 가장 효율적인 공부 방식

추천 공부 흐름:

기출 풀이
↓
유형 태그 달기
↓
분류 트리 연결
↓
예외 정리
↓
함정 기록

핵심은:
- 문제를 “외우는 것”이 아니라
- “어떤 유형인지 즉시 분류 가능하게 만드는 것”.
